"рђЪрђг рђЪрђ»рђљрђ║рђюрђЙрђЂрђ╗рђірђ║рђюрђгрђИрђђрђйрђг рђђрђ▓ рђњрђФрђєрђГрђ» рђђрђГрђ»рђђрђ╗рђ▒рђгрђ║рђћрђгрђИрђЋрђ╝рђ«рђИ рђЏрђ▒рђЂрђ╗рђГрђ»рђИрђљрђ▒рђгрђирђЎрђџрђ║ рђЋрђ╝рђ«рђИрђЏрђёрђ║ рђАрђЏрђёрђ║рђћрђ▒рђирђђрђЋрђ╝рђ▒рђгрђЂрђ▓рђирђљрђ▓рђирђЁрђгрђљрђйрђ▒рђЋрђ╝рђћрђ║рђєрђђрђ║рђђрђ╝рђЎрђџрђ║ рђЪрђ»рђљрђ║рђЋрђ╝рђ«рђюрђгрђИ''

рђђрђГрђ»рђђрђ╗рђ▒рђгрђ║рђљрђГрђ»рђирђірђ«рђђрђГрђ» рђЏрђ▒рђЂрђ╗рђГрђ»рђИ рђЉрђЎрђёрђ║рђИрђЁрђгрђИрђАрђЋрђ╝рђ«рђИ

"рђђрђ▓ рђЄрђ▒рђгрђ║рђЄрђ▒рђгрђ║ рђЪрђГрђ»рђћрђ▒рђирђђрђЋрђ╝рђ▒рђгрђЉрђгрђИрђљрђгрђюрђ▒рђИрђљрђйрђ▒рђЎрђЙрђљрђ║рђЎрђГрђърђ▒рђИрђљрђџрђ║рђЎрђюрђгрђИ рђЎрђ▒рђирђЏрђёрђ║рђЋрђ╝рђћрђ║рђЋрђ╝рђ▒рђгрђЦрђ«рђИрђћрђ▒рђгрђ║''

"рђЪрђ»рђљрђ║рђђрђГрђ»рђђрђ╗рђ▒рђгрђ║ рђЎрђЙрђљрђ║рђЎрђГрђЋрђФрђљрђџрђ║''

"рђђрђ▓ рђњрђФрђєрђГрђ»рђєрђђрђ║рђЎрђџрђ║рЂІ рђњрђ«рђћрђ▒рђирђЋрђ╝рђ▒рђгрђЎрђЙрђгрђђ Forward Propagation(feed forward) рђћрђ▓рђи backward propagation рђєрђГрђ»рђљрђгрђЋрђ▓рЂІ рђЁрђђрђгрђИрђюрђ»рђХрђИрђАрђърђЁрђ║рђєрђГрђ»рђЋрђ╝рђ«рђИрђљрђ▒рђгрђирђЎрђюрђћрђ║рђирђърђйрђгрђИрђћрђ▓рђирђЦрђ«рђИрЂІ рђњрђФрђђрђЎрђёрђ║рђИрђЏрђёрђ║рђИрђћрђЙрђ«рђИрђЋрђ╝рђ«рђИрђърђгрђИрђљрђйрђ▒рђЋрђ▓рђђрђйрЂІ Technical terms рђљрђйрђ▒рђАрђћрђ▒рђћрђ▓рђирђЋрђ╝рђћрђ║рђЋрђ╝рђ▒рђгрђљрђгрђЋрђ▒рђФрђирђђрђйрђгрЂІ рђАрђЏрђёрђ║рђћрђ▒рђирђђрђЁрђгрђљрђйрђ▒рђюрђ▓рђЋрђ╝рђћрђ║рђћрђйрђЙрђ▒рђИрђЏрђёрђ║рђИрђЋрђ▒рђФрђирЂІ''

"рђЪрђ»рђљрђ║ рђђрђГрђ»рђђрђ╗рђ▒рђгрђ║ рђћрђ▓рђћрђ▓рђљрђ▒рђгрђёрђ║рђюрђћрђ║рђирђърђйрђгрђИрђљрђџрђ║рђЪрђ▓рђЪрђ▓''

"Feed forward рђђрђўрђгрђюрђ»рђЋрђ║рђюрђ▓рђєрђГрђ»рђљрђ▒рђгрђи рђђрђГрђ»рђђрђ╗рђ▒рђгрђ║рђљрђГрђ»рђиNeuron Network рђЉрђ▓рђЎрђЙрђгрђЏрђЙрђГрђърђЎрђ╗рђЙ neuron рђљрђйрђ▒рђђрђГрђ» data рђљрђйрђ▒рђЉрђірђ║рђирђљрђџрђ║ рђАрђЉрђ▓рђЎрђЙрђг activation function рђљрђйрђ▒рђћрђ▓рђирђљрђйрђђрђ║рђљрђџрђ║ рђЋрђ╝рђ«рђИрђЏрђёрђ║output рђЉрђ»рђљрђ║рђЋрђ▒рђИрђљрђџрђ║рЂІ рђАрђљрђГрђ»рђЂрђ╗рђ»рђЋрђ║рђєрђГрђ»рђЏрђёрђ║ рђЮрђёрђ║рђюрђгрђљрђ▓рђи data рђљрђйрђ▒рђђрђГрђ» prediction рђюрђ»рђЋрђ║рђљрђџрђ║рђЋрђ▒рђФрђирђђрђйрђгрЂІ рђЋрђ╝рђ«рђИрђЏрђёрђ║ error рђљрђйрђђрђ║рђљрђџрђ║ рђўрђџрђ║рђюрђГрђ»рђљрђйрђђрђ║рђюрђ▓рђЎрђЙрђљрђ║рђЎрђГрђърђ▒рђИрђюрђгрђИ''

"рђЪрђ»рђљрђ║рђђрђГрђ»рђђрђ╗рђ▒рђгрђ║ desired output - actual output''

(рђЏрђЙрђ▒рђи post рђЎрђЙрђг target output рђюрђГрђ»рђирђЏрђ▒рђИрђЉрђгрђИрђљрђгрђћрђ▓рђи рђАрђљрђ░рђљрђ░рђЋрђФрђЋрђ▓рђЂрђёрђ║рђЌрђ╗)
