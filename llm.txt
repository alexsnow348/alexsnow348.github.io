# Alex Snow Learning Journey (လေ့လာမှု ခရီးစဥ်များ)

## Overview
A personal learning journey blog documenting life experiences, thoughts, and lessons learned. This site is a collection of writings in Burmese (Myanmar) and English, created to reflect upon later in life.

**Tagline:** Alex Snow Learning Journey  
**URL:** https://alexsnow348.github.io  
**Author:** Alex Snow  
**Languages:** Burmese (Myanmar), English  

## Site Description
ဘဝတစ််လျှောက် လေ့လာမိသမျှ၊ လုပ်မိသမျှ၊ တွေးမိသမျှတွေကို နောက်ပိုင်း အသက်ကြီးတဲ့အချိန် ရောက်လာတဲ့အခါ ပြန်လည်ဖတ်ပြီး ပြုံးပျော်နိုင်ဖို့ မှတ်တမ်းတင်ထားတာတွေ စုထားတဲ့ နေရာ‌လေး ဖြစ်ပါတယ်။

(Translation: A place where I collect all the things I've learned, done, and thought throughout my life, so that I can read them back and smile when I get older.)

## Technical Details

### Technology Stack
- **Platform:** Jekyll static site generator
- **Hosting:** GitHub Pages
- **Markdown:** Kramdown
- **CSS:** SCSS/Sass
- **Pagination:** 3 posts per page
- **Plugins:**
  - jekyll-paginate
  - jekyll-sitemap
  - jekyll-seo-tag
  - jekyll-font-awesome-sass

### Site Structure
- `_posts/`: Blog posts in Markdown format (YYYY-MM-DD-title.md)
- `_layouts/`: Page templates (default, post, page)
- `_includes/`: Reusable components (head, sidebar)
- `_scss/`: Styling files
- `public/`: Static assets (images, fonts, CSS)
- `blog/`: Blog listing pages
- `drafts/`: Work-in-progress posts

### Post Naming Convention
Posts follow Jekyll's standard naming: `YYYY-MM-DD-post-title.md`

Posts can be in English or Burmese (Myanmar) language.
